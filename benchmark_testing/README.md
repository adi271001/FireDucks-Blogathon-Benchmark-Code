# AI & Data Science Blogathon - Benchmarking DataFrame Libraries

This repo contains benchmark analysis and visualizations comparing Pandas, FireDucks, Polars, and Modin on ETL tasks.

## Folder Structure
- `benchmarks/`: Tool-specific benchmarking scripts (coming soon)
- `plots/`: Generated benchmark visualizations
- `data/`: Synthetic dataset or CSVs
- `main.py`: Visualization code
- `requirements.txt`: Environment dependencies

## Run the Code
```bash
pip install -r requirements.txt
python main.py
```
